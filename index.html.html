01 — Machine Learning

Air Quality Prediction with PySpark

Air pollution data is massive and messy. The challenge was to build a pipeline that could process large environmental datasets efficiently and generate reliable predictions — something a single machine couldn't handle comfortably.

Python PySpark MLlib Distributed Computing
  • Built an end-to-end ML pipeline — from raw data ingestion to model evaluation — using PySpark's distributed framework
  • Performed feature engineering on large-scale environmental datasets to improve prediction accuracy
  • Optimized processing throughput by leveraging distributed computation instead of single-node approaches
  • Visualized trend analysis insights that could inform data-driven environmental decisions
Takeaway — Working with PySpark shifted how I think about scalability. When your dataset outgrows a single machine, the architecture has to change — and designing for that is a different kind of challenge.

02 — Hackathon · Full-Stack

Digital Health Record Wallet

Medical records are scattered, inaccessible, and often insecure. This hackathon project tackled the problem of giving individuals a simple, secure way to store, manage, and share their health data — without friction.

Next.js React Tailwind CSS Firebase
  • Designed and shipped a responsive full-stack web application within hackathon time constraints
  • Built a clean, intuitive UI for uploading, categorizing, and retrieving health records securely
  • Integrated Firebase for real-time database operations and authentication workflows
  • Implemented structured data handling with basic access controls to protect sensitive health information
Takeaway — Shipping under pressure taught me how to prioritize ruthlessly. Not every feature makes it to v1 — knowing what the core experience needs to feel complete is a skill in itself.
